# ClickFlow v3 - MERN Stack

## Architecture
- Backend: Node.js + Express.js + Mongoose (converted from Python/FastAPI)
- Frontend: React + Tailwind + Shadcn
- Database: MongoDB
- Auth: JWT httpOnly cookies + bcryptjs

## Implemented
- [x] Full MERN backend (8 Mongoose models, 7 route files)
- [x] Sprint Board Kanban, QA Bug Tracking, Cross-project views
- [x] Comments, Activity log, Notifications, Attachments
- [x] Light/Dark theme, Search, Role-based access
- [x] README with local setup guide + API docs

## Testing: 100% backend (32/32), 95% frontend
